# react-native-test-task
Test task for the intership. Calculator in react native

Simple calculator app. 

Простий калькулятор підтримує обчислення між двома числами та прості математичні операції, 
вміє зберігати число у пам'яті та відтворювати його. 

m+ - додати (+) у пам'ять значення числа, що знаходиться на екрані

m- - відняти (-) з пам'яті значення числа, що знаходиться на екрані

mr - відобразити на екрані значення, що записано у пам'яті на даний момент

mc - очистити значення, що записано у пам'яті (стає "0")

Повторне натискання на знак математичної операції скидує значення попереднього введеного числа до 0, 
залишаючи змогу провести операцію заново і більш точно.
Калькулятор не дає записувати більше одного знака коми у дробове число.
